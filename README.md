## Development

Install the dependencies:

```sh
bundle install
```

Then serve the site and compile the CSS:

```sh
# First tab, this will serve the site at http://127.0.0.1:4000/
bundle exec jekyll serve

# Second tab
sass --scss --style compressed --watch assets:assets
```
